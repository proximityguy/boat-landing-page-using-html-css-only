# Tech Shop Website

This project is a responsive technology shop website built using HTML5 and CSS3. It showcases various tech products with an emphasis on modern web design practices.

## Features

- **Responsive Design**: Ensures optimal viewing experience across various devices and screen sizes.
- **Product Showcase**: Displays a range of tech products with images and descriptions.
- **Clean Layout**: Utilizes modern CSS techniques for a clean and user-friendly interface.

## Project Structure

- `images/products/`: Directory housing product images used throughout the website.
- `source/`: Contains the main HTML and CSS files that constitute the website's structure and styling.
- `Readme.txt`: Provides basic information about the project.
